# 极简影院

> 使用[Vue](https://cn.vuejs.org/)和[iview](https://iviewui.com/)开发的一个简单的观影系统


## 安装的依赖

``` bash
# vue-router
# hls.js
# axios
# x2js
# dplayer
# social-share.js
```

## 小结
一个简单的影院系统，借助网络上的资源，使用后台抓取数据返回给前台来实现视频搜索播放。
